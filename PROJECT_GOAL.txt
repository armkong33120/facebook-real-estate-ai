# OBJECTIVE: GHOST AGENT - AI VISION-FIRST FACEBOOK SCRAPER (V36.00+)

จุดประสงค์หลักของโครงการนี้คือการปฏิวัติระบบการดึงข้อมูลทรัพย์ (Real Estate) จาก Facebook โดยเปลี่ยนจากวิธีเดิมที่เป็น DOM-Based (การหา Selector ที่เปลี่ยนแปลงบ่อย) มาสู่ระบบ "AI Vision-First" ที่มีความเสถียรสูงบนเครื่อง Mac ของคุณ

## 🎯 เป้าหมายโดยละเอียด:

1.  **AI Vision Monitoring (ดวงตา AI)**: 
    ใช้ AI (Gemini Vision) ทำหน้าที่แทนดวงตาคน โดยการสแกน Screenshot เพื่อระบุพิกัด (X, Y) ที่จะคลิกจริงบนจอภาพ ทำให้ระบบไม่พังเมื่อ Facebook เปลี่ยนโครงสร้าง HTML

2.  **Step-by-Step Control (การควบคุมแบบขั้นบันได)**:
    แบ่งการทำงานออกเป็น 6 ขั้นตอน (Step 1-6) เพื่อให้ง่ายต่อการทดสอบและปรับแต่ง:
    - Step 1: เปิด Browser (Chrome for Testing) แบบอิสระ (Launch & Detach)
    - Step 2: การเข้าถึงหน้าเป้าหมายและการจัดการ Login/Session
    - Step 3: การสแกนหาโพสต์ทรัพย์ด้วยสายตา AI
    - Step 4: การคลิกเปิด Lightbox และเลื่อนรูปภาพด้วยพิกัดจริง
    - Step 5: การดึงรูปภาพคุณภาพสูงและการตรวจสอบ Image Hash (กันรูปซ้ำ)
    - Step 6: การบันทึกข้อมูลและส่งออกผลลัพธ์ที่สมบูรณ์

3.  **Hardware Consistency (ความแม่นยำของพิกัด)**:
    ล็อคขนาด Viewport ไว้ที่ 1440x900 (Logical Resolution) เพื่อให้พิกัดที่ AI วิเคราะห์ออกมา ตรงกับจุดที่เมาส์จะคลิกในหน้าจอ Mac ของผู้ใช้ 100%

4.  **Human Stealth (การพรางตัวแบบมนุษย์)**: 
    ใช้ Google Chrome for Testing พร้อมโปรไฟล์จริง และใช้เทคนิคซ่อนแถบ Automation เพื่อลดความเสี่ยงในการถูกตรวจจับ

## 🛠️ โครงสร้างไฟล์ปัจจุบัน:
- config.py: ศูนย์รวมการตั้งค่าและ API Key
- ai_engine.py: ระบบเชื่อมต่อดวงตา AI (Gemini API)
- browser_core.py: ระบบควบคุมการเปิดสถาปัตยกรรมบราวเซอร์แบบอิสระ
- ghost_main.py: ตัวสั่งรันเรียงลำดับขั้นตอน (Step 1 -> 6)
- step1.py - step6.py: ไฟล์รองรับตรรกะในแต่ละขั้นตอนที่กำลังจะพัฒนา

## 🚀 ผลลัพธ์ที่เราคาดหวังจะได้รับ (EXPECTED OUTCOMES):

1.  **ความแม่นยำสูง (High Precision)**: 
    สามารถเก็บภาพทรัพย์ได้ครบถ้วนและเป็นภาพที่มีคุณภาพสูงที่สุด (HQ) จาก Facebook โดยไม่มีความผิดพลาดจากการหา Selector ไม่เจอ

2.  **ระบบไร้รูปซ้ำ (No Duplication)**: 
    รูปภาพทุกลูกที่ถูกบันทึกจะผ่านการตรวจสอบ Image Hash 100% เพื่อมั่นใจว่าในอัลบั้มของคุณจะไม่รูปที่ซ้ำกันเลยแม้แต่รูปเดียว

3.  **ค่าบำรุงรักษาต่ำ (Low Maintenance)**: 
    ระบบจะไม่พังเวลา Facebook อัปเดต UI เล็กๆ น้อยๆ เพราะ AI Vision จะยังคงหาปุ่มและรูปภาพได้เหมือนเดิม (เหมือนคนมอง)

4.  **ความเสถียรในการทำงานต่อเนื่อง (Batch Stability)**: 
    สามารถรันงานต่อเนื่องหลักพันรายการ (ตามไฟล์ uat_links.txt) ได้โดยไม่ต้องเฝ้าสังเกตการณ์ตลอดเวลา

5.  **การพรางตัวที่สมบูรณ์ (True Stealth)**: 
    Facebook จะมองไม่ออกว่าเป็นโปรแกรมอัตโนมัติ เพราะพฤติกรรมทั้งการคลิก การรอ และการใช้พิกัดหน้าจอ จะเหมือนมนุษย์ที่ใช้งาน Chrome ปกติทุกประการ
